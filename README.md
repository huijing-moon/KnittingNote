# 🧶 KnittingNote

> 내 손뜨개 프로젝트를 기록하고 관리하는 iOS 앱 (SwiftUI)

KnittingNote는 뜨개질 프로젝트를 체계적으로 기록하고,  
단수 진행 상황과 사진, 메모까지 쉽게 관리할 수 있는 iOS 앱입니다.  

---

## 🎨 주요 기능

- 프로젝트별 관리
  - 프로젝트 제목, 실 종류, 바늘 정보 기록
  - 현재 진행 단수(Row) 기록
- 사진 첨부
  - 프로젝트별 사진 추가
- 메모 관리
  - 단수별 메모 작성 가능
- 직관적인 카드형 UI
  - 프로젝트 목록 한눈에 보기
  - 부드러운 파스텔톤 디자인
